# Release Notes

## v0.1.0

Initial public release of `company-growth-check-in`.

### What is included

- a reusable skill for evaluating whether a company or role is still worth staying in
- bootstrap flow for collecting company, role, and user-goal context
- agent-led discussion flow that starts with questions instead of analysis
- daily report writing for dated decision records
- weekly and monthly review templates for pattern-based reflection
- runtime privacy defaults through ignored `context/` and `reports/` files

### Who it is for

- people in early-stage companies
- people trying to judge whether a role is still giving enough learning return
- people who want to turn messy day-to-day work signals into clearer career decisions

### Release focus

This first release focuses on one job:

Determine whether the current company or role is still worth staying in, and whether the user is still getting meaningful growth from staying.
